# Фабрика цифровых двойников - тренажёр

Игровой веб-тренажёр по курсу ПИШ СПбПУ «Цифровые двойники изделий»: 16 модулей курса = 16 шагов
сборки цифрового двойника выбранного изделия (авиадвигатель/ГТД, автомобиль, БПЛА, ТВС, турбина,
нефтегаз), мини-механики вместо тестов, проверка знаний, очки/звёзды/рейтинг, финальный
виртуальный запуск двойника.

## Как открыть
**По ссылке (ничего ставить не нужно):** после включения GitHub Pages тренажёр доступен по адресу
`https://<логин>.github.io/<имя-репозитория>/` - просто откройте в браузере.

**Локально:** кнопка **Code → Download ZIP**, распаковать, открыть **`index.html`** двойным кликом.
Сервер не нужен - тренажёр статичный.

## Что внутри
- `index.html`, `app.js`, `app.css` - приложение (чистый JS, без сборки).
- `content/` - банк вопросов (курсовой + отраслевой).
- `assets/` - арт: баннеры модулей и стендов по 6 изделиям, видео, иконки, логотипы.

Учебный тренажёр. Терминология - по ГОСТ Р 57700.37 и материалам курса ПИШ СПбПУ.
